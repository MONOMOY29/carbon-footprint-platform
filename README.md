# Carbon Footprint Awareness Platform

## Challenge
Hack2Skill Virtual PromptWars - Challenge 3

## Vertical
Individual Carbon Footprint Tracker

## Approach
A React web app where users input daily activities (transport, food, energy) and get their estimated carbon footprint in kg CO₂, along with personalized tips to reduce it.

## How It Works
1. User enters daily km travelled, meals eaten, and kWh used
2. App calculates total CO₂ using emission factors
3. Displays impact level (low/moderate/high)
4. Shows personalized reduction tips per category

## Assumptions
- Transport factor: 2.5 kg CO₂/km
- Food factor: 1.8 kg CO₂/meal
- Energy factor: 1.2 kg CO₂/kWh
